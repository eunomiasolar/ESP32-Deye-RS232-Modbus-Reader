ESP32 Deye RS232 Modbus Monitor

</div>


This is a fully usable App that display Deye RS232 Modbus registers via WiFi on a browser 
and generate a “LogYY-MM-DD_hh_mm.csv” file with entries every 10 seconds and transfer 
this 0.5 GB file via FTP to a PC ftp server that run somewhere on the WiFi network.
